# OpenGL 2D PBR renderer expiriment

A simple demo of a simplistic 2D PBR renderer written in C++ and utilizing OpenGL.

### Features
- Normal mapping
- Roughness mapping (Specular lighting)
- Ambient Occlusion mapping
- Textured lights
